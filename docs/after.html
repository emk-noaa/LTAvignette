To fix this we will stage a similar edit, this time copying and pasting two rows (P and V events) below the rogue R event:

edit_1607_68 <- 
  list(das_file = das_file, 
       type = 'copy',
       rows = c(129982, 129985),
       chars = NULL,
       edit = 129987)
